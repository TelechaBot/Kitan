```dotenv
VITE_CLOUDFLARE_SITE_KEY=0x4AAAxxxxxxxxx
# Cloudflare Turnstile
```

## Recommended Setup

```shell
cd app
npm install
env VITE_BACKEND_URL="https://api.example.com"
# env VITE_CLOUDFLARE_SITE_KEY="optional"
npm run build
# Deploy to server
```
